Download

BayesX is available for Windows (95/98/2000/NT/Xp). Until version 1.30, BayesX was distributed in two different versions, a Java based version and a non Java based version. Starting with version 1.40, the non Java based version of BayesX is no longer supported, since the Java based version has additional functionality.

Download

Last Update: Version 1.40 (18.07.2005)

Download the complete package (incl. manuals and examples) by clicking on the 'download' button below. Store the file installBayesX.exe in a temporary directory. Execute the program and follow the installation instructions on the screen. After a successful installation, BayesX can be executed using the Windows start button. The manuals of the program can be found in the 'doc' directory which is a subdirectory of the installation directory. They are also available from the BayesX help menu.

Manuals

BayesX is distributed with three manuals: A reference manual containing a detailled description of the BayesX commands, a methodology manual containing a short review of the methodological background, and a tutorial manual containing tutorial like sections for new users. All three manuals are part of the standard installation of BayesX and can be found in the 'doc' directory which is a subdirectory of the installation directory. They are also available from the BayesX help menu.

Registration

If you want to register yourself as a BayesX-user, please enter your E-Mail-address below and click on the 'submit' button. Registration is not necessary for using BayesX, but as a registered user you will be informed about future releases of BayesX or if severe bugs have been found in the current version.

Problems with the current version

Please obey the following limitations:

  • Cumulative threshold models are restricted to three categorical responses (bayesreg objects only).
  • The manual describes for first and second order random walk priors automatic determination of block sizes for block move updates. This it not yet implemented. For P-splines, however, the automatic determination of block sizes works properly.

The bug list contains a complete list of (severe) bugs of the different versions of BayesX.
