Download

BayesX is available for Windows (95/98/2000/NT/Xp). To save download time, most files are compressed using Winzip. Currently we distribute two versions of BayesX, a Java based version and a non Java based version. The Java based version has additional functionality compared to the non Java based version. In particular, the Java based version has graphics facilities.

You may register yourself as a BayesX-user. In this case you will be informed when new versions are released or if severe bugs have been found.

We recommend to download and use the Java based version although this version runs slightly slower than the non Java based version. Please choose from the pull-down menu below.

Java based Version

Last Update: Version 1.30 (08.11.2004)

  • Download the complete package (incl. manual and examples). To install BayesX on your Computer, select "Java based version" from the pull down menu below and click on the 'download' button. Store the file installBayesX.exe in a temporary directory. Execute the program and follow the installation instructions on the screen. After a successful installation, BayesX can be executed using the Windows start button. The manual of the program (file manual.pdf) can be found in the 'doc' directory which is a subdirectory of the installation directory.

Non Java based Version

Last Update: Version 1.30 (08.11.2004)

  • Download the exe-file only. This is useful, if BayesX has already been installed on your computer. In this case it is faster to download the exe-file only, and replace the old exe-file on your computer with the new version. Select "Exe-file for the non java based version" from the pull down menu below and click on the 'download' button.
  • Download the complete package (incl. manual and examples). To install BayesX on your Computer, select "Non java based version" from the pull down menu below and click on the 'download' button. Store the file bayesx.zip in a temporary directory. Unzip the file using Winzip. Execute the program setup.exe and follow the installation instructions on the screen. After a successful installation, BayesX can be executed using the Windows start button. The manual of the program (file manual.pdf) can be found in the 'doc' directory which is a subdirectory of the installation directory.

Problems with the current version

Please obey the following limitations:

  • Cumulative threshold models are restricted to three categorical responses.
  • The manual describes for first and second order random walk priors automatic determination of block sizes for block move updates. This it not yet implemented. For P-splines, however, the automatic determination of block sizes works properly.
  • In the Java based version, the generation of variables works properly only if random numbers are not involved (e.g. generate x = uniform() works not properly). In the non Java based Version, however, the problem does not occur.

The bug list contains a complete list of (severe) bugs of the different versions of BayesX.
