Tutorials

To make the user familiar with the usage and the capabilities of BayesX we present some example analyses which may serve as first introductions. Each tutorial is designed to be self contained, that is, each tutorial gives a short introduction to all features of BayesX you will need throughout the tutorial.

Data sets and map objects that are needed to execute the commands presented in the tutorials as well as batch files containing these commands may be downloaded from the corresponding sections below.
