Erstellen einer neuen BayesX Installation:



HANDBUCH


1. Im Handbuch Versionsnummern ändern (in manual_preliminaries.tex).

2. Im Handbuch Literatur checken.

3. Handbuch in pdf umwandeln.


JAVAVERSION

4. In BayesX.java die Versionsnummern ändern (3 mal).

5. Projekt BayesX_project.bpg im C++ Builder öffnen. Alle dlls compilieren.

6. In der Dos Konsole:
    
   cd c:\bayesx\java
   c:\programme\jdk\bin\javac BayesX.java

7. java BayesX - Default windowing herstellen

8. Checken, ob in  c:\bayesx\java\temp und c:\bayesx\java\temp
   keine Dateien sind. Gegebenenfalls entfernen. 

9. Installationsroutine java herstellen. Funktioniert nur wenn Schreibrechte für 
   bayesxjava.iap_xml vorhanden sind!!!


C++ Version

10. Projekt BayesX.bpr compilieren

11. Installationsroutine erstellen

12. exe Datei unter bayesxexe.zip zippen, setup Dateien unter bayesx.exe zippen


INTERNETSEITEN


10. Literatur in Internetseiten checken (bayesxwhat.html).

11. Versionsnummern ändern  (in bayesx.html, bayesxdownload.html (2 mal),) 

12. New features hinzufügen (in bayesxnews.html)

13. Bug Seite ändern (bayesxbugs.html)
